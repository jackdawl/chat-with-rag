# LlamaIndex RAG 实战项目

## 项目概述

这是一个基于 LlamaIndex 框架构建的RAG系统，提供了完整的文档处理、检索和对话功能。
该系统支持多种格式的文档（PDF、TXT、DOCX、Markdown）上传、处理，并通过混合检索策略（向量检索 + BM25）提供高质量的问答服务。
